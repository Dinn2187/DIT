1. Переходим в директорию проекта и определяем текущую директорию, как директорию репозитория (git init)


2. Добавляем файлы в наш новый локальный репозиторий, чтобы добавить файлы в первый коммит (git add .)


3. Коммит файлов (git commit -m "First commit")


4. Далее копируем ссылку/SSH-key на сам репозиторий


5. В командной строке добавляем ссылку на удаленный репозиторий ( 
- Указание на новый удаленный репозиторий:    git remote -v
- Проверка нового удаленного адреса:     git remote add origin ссылка на удаленный репозиторий или SSH-key git@example.com:example/test.git)


6. Отправляем изменения локального репозитория в удаленный, помеченный как источник (git push origin master).
